Project :

‘Plat.home’- Concept, UX Design and Interaction Guide
for Samsung Smart TV


Role :

User Research, UX Designer, GUI Designer


Description :

As the video contents channels had been diversed, the role of Smart TV needs to be repositioned. So we defined smart TV as an integrated contents platform including not only multimedia contents but also IoT services consumed by users at home. Also based on the heuristic principles, we designed hardware interfaces on the remote controller matching with the user interface of the Smart TV.


Output :

UX strategies, Interface design and prototypes with guides, 3D printed Hardware Mock-up
